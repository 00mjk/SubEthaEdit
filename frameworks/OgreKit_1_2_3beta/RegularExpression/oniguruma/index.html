Oniguruma

2004/03/16 --- (C) K.Kosako

Oniguruma is a regular expressions library.
The characteristics of this library is that different character encoding for every regular expression object can be specified.

Supported character encodings:

ASCII, UTF-8, EUC-JP, EUC-TW, EUC-KR, EUC-CN, Shift_JIS, Big5, KOI8, KOI8-R, ISO-8859-1, ISO-8859-2, ISO-8859-3, ISO-8859-4, ISO-8859-5, ISO-8859-6, ISO-8859-7, ISO-8859-8, ISO-8859-9, ISO-8859-10, ISO-8859-11, ISO-8859-13, ISO-8859-14, ISO-8859-15, ISO-8859-16

There are two ways of using of it in this program.

  • C library (supported APIs: GNU regex, POSIX, Oniguruma native)
  • Built-in regular expression engine of Ruby 1.6/1.8
    In Ruby 1.9, Oniguruma is already incorporated by Kazuo Saito.
