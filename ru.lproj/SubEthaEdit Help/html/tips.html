Советы
 

Чтобы дать команду SubEthaEdit из терминала добавьте alias see='open -a /Applications/SubEthaEdit.app' в .bashrc

Если у вас запущен SubEthaEdit Server вы можете добавить функцию автосохранения osascript -e 'save documents of application "SubEthaEdit"', который будет работать как крон.

Вы можете добавлять на вашем сайте ссылки на документы SubEthaEdit с помощью <a href="see://hostname/document.txt">.

Если после завершения совместного редактирования документа вы хотите разослать финальный текст всем, щелкните с нажатым Control по выбранной группе участников в полке или выберите команду в контекстном меню. Скопируйте текст в письмо и готово.

Чтобы разрешить пользователю подключиться к совместной работе над документом, перетащите его пиктограмму из полки участников прямо в документ. Эта функция работает и в режиме раздельных окон.
