Tips
 

To call SubEthaEdit from Terminal add alias see='open -a /Applications/SubEthaEdit.app' to your .bashrc

If you are running a SubEthaEdit Server you can add autosave by calling osascript -e 'save documents of application "SubEthaEdit"' as a cronjob.

You can link to SubEthaEdit documents by placing <a href="see://hostname/document.txt"> on your website.

If you are hosting a collaboration session and want to email the text to all users after having finished, select them in the participants drawer and control-click or use the action menu. Copy the text to the mail client and there you go!

To follow a user in collaboration mode drag them from the participants drawer into the document. This feature is great combined with the splitview.
