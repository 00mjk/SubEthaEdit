Aperçu
 

Dans ce chapitre vous trouverez des informations sur SubEthaEdit.

Partager des documents
Pour autoriser des utilisateurs à voir les documents que vous souhaitez partager, vous devez cliquer sur le bouton Partager de la barre d'outils ou presser les touches ⇧⌘E. Ceci fait, ils verront votre document dans leur gestionnaire Bonjour et pourront s'y connecter en double-cliquant dessus.

Invitations
Pour inviter une personne à rejoindre votre document, glissez là dans votre document depuis le gestionnaire Bonjour ou Internet.

Rejoindre des documents
Si vous souhaitez rejoindre un document de votre sous-réseau, double-cliquez juste dessus depuis le gestionnaire Bonjour (⌘K). Si vous souhaitez rejoindre un document Internet, ouvrez le gestionnaire Internet (⇧⌘K) et entrez-y le nom d'hôte ou l'adresse IP de la personne partageant le document.

Visible contre Invisible.
Si vous souhaitez que les autres utilisateurs puissent vous inviter, vous devez choisir "Toujours visible" dans le menu contextuel du gestionnaire Bonjour. Pour plus d'intimité, vous pouvez choisir "Visible en cas d'annonce".
