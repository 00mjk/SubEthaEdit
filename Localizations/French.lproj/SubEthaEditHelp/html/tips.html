Astuces
 

Pour appeler SubEthaEdit depuis le Terminal, utilisez le nouvel outil de la ligne de commande : /usr/bin/see

Si vous gérez un serveur SubEthaEdit, vous pouvez gérer des sauvegardes automatiques en utilisant osascript -e 'save documents of application "SubEthaEdit"' comme tâche cron.

Vous pouvez faire un lien vers vos documents SubEthaEdit en rajoutant <a href="see://adresseIP/document.txt"> sur votre site web.

Si vous hébergez une session collaborative et souhaitez envoyer le texte par e-mail une fois celui-ci terminé ; séléctionnez les participants dans le gestionnaire et controle-cliquez ou utilisez le menu d'actions. Copiez le texte dans votre client mail et c'est tout !

Pour suivre un utilisateur en mode collaboratif, glissez le depuis la liste des participants vers le document. Cette fonctionnalité est encore plus agréable lorsqu'elle est couplée avec la vue séparée.

Pour conserver les couleurs et la mise en forme d'un texte copié (par exemple pour le coller dans un document RTF), utilisez la commande "Copier avec les styles" du menu Édition ou du menu contextuel.
