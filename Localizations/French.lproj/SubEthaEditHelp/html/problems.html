Résolution des problèmes
 

Si vous avez besoin d'aide pour résoudre un problème lié a l'utilisation de SubEthaEdit, consultez les éléments ci-dessous.

J'utilise le seul Macintosh de mon sous-réseau. Où puis-je tester le mode collaboratif ?
Connectez vous au serveur officiel CodingMonkeys en cliquant ici.

SubEthaEdit 2.0 est-il compatible avec les versions 1.x ?
Non. Due à son très évolué nouveau protocole réseau, SubEthaEdit 2.0 ne peut pas communiquer avec les versions 1.x. Cependant le nouveau protocole est très souple et supportera les futurs développements.

Je ne vois aucun document partagé. Quel est le problème ?
Vérifiez d'abord l'utilisation d'un pare-feu (Firewall). Si c'est le cas, vous devrez autoriser SubEthaEdit à utiliser les ports 6942 et 6951. Si vous utilisez le pare-feu intégré à Mac OS X, vous devez suivre ces étapes : Ouvrir les préférences systèmes → Partage → Coupe-feu. Cliquez sur le bouton "Nouveau...", choisissez Autre et entrez "6942-6951" pour le numéro de port et "SubEthaEdit" pour la description.

Je suis derrière un routeur. Comment puis-je travailler avec d'autres utilisateurs : eux aussi derrière un routeur ?
L'un des utilisateurs doit activer le transfert de port sur son routeur. Les ports 6942 et 6951 doivent être redirigés du routeur vers l'ordinateur. Ensuite, les autres utilisateurs devraient pouvoir établir une connexion qui sera utilisée pour le partage dans les 2 sens. Si vous avez besoin de connaître votre adresse IP dynamique, vous pouvez essayer http://whatismyip.com
