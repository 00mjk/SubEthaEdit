새로운 기능
 

SubEthaEdit 2.1로 다음과 같은 것을 하실 수 있습니다:

커맨드 라인 툴
새로운 커맨드 라인 툴은 터미널 응용 프로그램으로 작업하려는 분들께 완벽한 도구가 될 것입니다. 파이핑 등 모든 익숙한 것들을 할 수 있도록 지원합니다. 예를 들어, curl codingmonkeys.de | see -m HTML와 같은 것들 말입니다. 또는 로그 등을 기록하게 하기 위해서 SubEthaEdit를 $EDITOR로 지정할 수 있도록 합니다. (export EDITOR='see -wr') 모든 기능에 대한 자세한 설명을 보려면, 커맨드 라인에서 man see을 입력하세요.

새롭게 변신한 인쇄 기능
이 버전에서 인쇄 기능이 매우 향상되었습니다. 다른 기능들처럼 인쇄 기능도 환경 설정에서 형식별로 사용자화가 가능합니다. 그러므로 일반 텍스트와 C 코드에 다른 형식을 지정할 수 있는 것입니다. 또한 인쇄 윈도우에서 SubEthaEdit 패널을 통해 인쇄 상태를 사용자화할 수도 있습니다. 날짜나 파일 이름과 같은 기본적인 기능에 더불어, 여백과 서체 그리고 색상, 참가자의 목록과 저작 정보, 배경의 색상 등까지 다양한 인쇄 정보를 설정할 수 있습니다.

HTML로 보내기
HTML로 보내기는 두 가지 방법으로 제공됩니다. 저작 정보를 포함하여 해당 문서의 협력 작업을 저장하기 위해 문서 전체를 HTML로 보내거나, "편집/XHTML로 베꺼두기" 기능을 통해 선택된 코드를 색상, 들여쓰기 등의 정보와 함께 HTML로 베꺼서 웹사이트에 붙여넣기위해 변환할 수 있습니다.

관리자 권한을 통한 열기와 저장하기
SubEthaEdit 2.1는 시스템 관리자로 권한으로만 편집할 수 있는 파일에 대한 접근을 위해 인증 절차를 거칠 수 있게 합니다. (예를 들어 /etc/httpd/httpd.conf와 같은 파일)

신택스 스타일
신택스 스타일은 어떤 형식이든지 원하는대로 색상/서체/스타일을 바꿀 수 있도록 합니다. 스타일을 따로 저장하여 친구나 동호회에서 공유할 수 있도록 합니다. 사용자화는 SubEthaEdit/환경설정/스타일에서 편리한 WYSIWYG 방식으로 할 수 있습니다.

개선된 AppleScript 지원
공유, 알리기, 접근 제어 등의 협력 작업 가능들이 이제 AppleScript로 사용 가능합니다. 또한 기본적인 편집 기능들에서도 좀더 개선된 Script를 지원합니다.

개선된 Tab 처리
이제는 포맷 메뉴에서 tab과 스페이스를 서로 변환할 수 있습니다. 또한 tab과 블록 편집 또는 들여쓰기와 같은 기능들을 개선하였습니다.

SubEthaEdit 2.0이 제공하는 기능들:

정규 표현식 검색과 대치
정규 표현식의 강력함으로 찾고 바꾸십시오. POSIX에서 Perl과 Ruby까지 8개의 정규 표현식 중에서 선택하십시오. 새로운 "모두 찾기" 함수를 통해서 새로 고침도 할 수 있는 검색 결과 목록을 사용하십시오.
정규 표현식에 대해 더 자세히 알아보기

새로운 협력 작업 기능 : 초대
이제 다른 사람들이 문서 작업에 들어올 때까지 기다릴 필요가 없습니다. 도움이 필요하다면, 사람들을 문서 안으로 드래그해서 초대하시면 됩니다. 1-2-3을 외치는 것만큼 쉽습니다.

공유 문서에 대한 읽기 전용 권한
사용자들이 해당 문서를 변경할 수 없는채로 볼 수 있게 하고 싶다면, 그들을 읽기 전용 권한으로 설정하십시오. 글을 볼 수는 있지만 바꿀 수는 없을 겁니다.

300% 더 효율적인 네트워크 프로토콜
새로운 네트워크 프로토콜을 따른 데이터 전송과 향상된 안정성 등의 새로운 기능을 갖췄습니다.

분할 윈도우
편집의 재미를 두배로 배가시키기 위해 여러분의 윈도우를 분할하실 수 있습니다!
