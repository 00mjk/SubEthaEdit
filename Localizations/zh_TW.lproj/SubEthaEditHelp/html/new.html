有什麼新功能
 

SubEthaEdit 2.1 新增的功能：

命令列工具
對於經常需要使用終端機來工作的使用者而言，新的命令列工具將是這些人的完美夥伴。這個命令列工具支援管線（piping）、等待（waiting）以及繼續已經停止的工作（resuming），所以使用者可以用它做出各種巧妙的操作，像是：curl codingmonkeys.de | see -m HTML，或是把 SubEthaEdit 設定成 $EDITOR 去讀寫文字檔，甚至是提交（commit）紀錄檔。（export EDITOR='see -wr'）欲知詳細的功能描述說明，請在命令列輸入man see。

全新的印列功能
印列功能在這次的版本中有了大幅度的改進。當中最重要的功能，就是可以自訂各種語法規則模式的印列設定，所以您可以為您的 C 程式碼調整不同的設定，而不再印出死氣沈沈的純文字檔。您也可以在 SubEthaEdit 的印列設定對話框中，自訂您的印出資訊。除了標準的功能像是加入標頭資訊（日期、檔名），邊界寬度與字體顏色外，您也可以指定印出協同編輯模式的後設資料，包含參與者清單，有註解的作者資訊以及背景顏色供彩色印列時使用。

輸出成 HTML
有兩種輸出成 HTML 的方式。你可以將整份文件輸出成 HTML，並且同時儲存協同編輯模式下所包含的作者資料。或者利用「編輯/複製成 XHTML」的功能，把您所選定的文字，包含顏色與縮排，統統轉成 HTML 格式，讓您公佈在您的網站上。

認證式的讀檔與存檔
SubEthaEdit 2.1 允許您以管理員身份的認證後，去編輯您原本沒有權限可以編輯的檔案。（像是 /etc/httpd/httpd.conf）

語法樣式
語法樣式允許您自訂任何一種語法規則模式的色彩、字體以及字體樣式。您甚至可以輸出您的樣式，與您的朋友或社群一同分享。這個位於偏好設定/樣式下的自訂功能，是以簡單直覺的原則設計出來，供使用者輕鬆上手。

更棒的 AppleScript 支援
協同編輯模式下的功能，像是共享，公佈，以及存取控制等等，現在都可以透過 AppleScript 來加以控制。我們也同時改進了 AppleScript 對基本編輯功能的支援。

改善對 Tab 的處理
您現在可以利用格式選單的選項，自由地在 Tab 與空白之間做轉換。此外，我們也修正了許多跟 tab 有關的問題與功能，像是區塊編輯與縮排的問題。

您可能不知道的 SubEthaEdit 2.0 新功能：

利用正規表示來尋找與取代
利用正規表示的強大功能來尋找與取代。您可以從八種不同（從 POSIX 到 Perl 和 Ruby）的正規表示語言中選出您最喜歡的來使用。全新的“尋找全部”功能則可以條列出您所找到的東西，並且及時更新尋找結果。
更進一步了解有關正規表示法的資訊

新的協同編輯功能: 邀請別人
現在您不需要再等待其他人加入您共享出來的文件了。如果您需要別人的幫助，直接（從 Rendezvous 或 Internet 瀏覽器）把使用者拖到文件中，即可以邀請對方參與。就跟數數字ㄧ樣簡單。

讓共享文件有唯讀功能
如果您不想參與者改變您的文件，但是還是希望他們能看到文件的變化，那就把他們設成唯讀狀態就可以了。簡言之，他們只能看，不能摸。

整整快了三倍的網路協定
新的網路協定擁有更多的功能，更快的資料傳輸率以及更好的擴充性。

分割顯示
分割您的文件視窗，讓您擁有兩倍的編輯樂趣！
