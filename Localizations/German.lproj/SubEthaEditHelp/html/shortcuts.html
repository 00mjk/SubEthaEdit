Tastatur-Kurzbefehle
 

Drücken Sie ⌥⎋ (alt-escape) oder F5 zum Aktivieren der automatischen Vervollständigung.

Halten Sie die Wahltaste gedrückt während Sie einen Text auswählen, oder drücken Sie ⇧⌘B nachdem Sie Text ausgewählt haben, um Blockedit zu aktivieren.

Drücken Sie ⎋ (escape) zum Beenden von Blockedit.

Drücken Sie ⌘R zum Anzeigen der Web-Vorschau.

Drücken Sie ⌘E zum Benutzen der Auswahl als Suchtext und drücken Sie anschließend ⌥⌘F zum Ausführen der "Alle Suchen"-Funktion.

Drücken Sie ⇧⌘I zum Anzeigen der unsichtbaren Zeichen im Text.
